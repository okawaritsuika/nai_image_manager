# NAI Image Manager v1.1.0 EXE

Windows 휴대용 EXE 버전에 GitHub 기반 자동 업데이트를 추가했습니다. 별도 업데이트 서버는 사용하지 않습니다.

## 주요 변경사항

- EXE 상단에 `업데이트 확인` 버튼을 추가했습니다.
- 새 버전이 있으면 GitHub Release에서 ZIP을 내려받아 SHA-256을 검증한 뒤 프로그램 파일만 교체합니다.
- 업데이트 후 NAI Image Manager가 자동으로 다시 실행됩니다.
- 업데이트에 실패하면 교체 전 프로그램 파일을 복구합니다.
- 갤러리 프롬프트 검색 인덱스 기능을 포함했습니다.

## 사용자 데이터 보존

다음 항목은 릴리스 ZIP에 포함되지 않으며 업데이트 시 삭제하거나 덮어쓰지 않습니다.

- `TOTAL_CLASSIFIED` 이미지 라이브러리
- `gallery_config.json`, `lab_config.json`, `quality_presets.json`
- `artists.json`, `styles.json`, `gallery_image_tags.json`
- `canvas_saved_setups.json`, `tag_dictionary_user_overrides.json`
- `data/tag_categories_ko.json`
- 출력·가져오기·작업 폴더와 그 밖의 사용자 파일

## 새로 설치하는 방법

1. `NAI_Image_Manager_v1.1.0-exe_windows.zip`을 다운로드합니다.
2. 원하는 위치에 ZIP 전체를 압축 해제합니다.
3. 폴더 안의 `NAI_Image_Manager.exe`를 실행합니다.

EXE만 따로 꺼내지 말고 압축 해제된 폴더 구조를 유지해 주세요.

## 기존 v1.0.0 Lite에서 이전하는 방법

1. 이 Release의 `NAIM_Updater_v1.1.0-exe.exe`를 기존 NAI Image Manager 폴더에 넣습니다.
2. 업데이터를 실행합니다.
3. 최신 프로그램 파일이 설치된 뒤 `NAI_Image_Manager.exe`가 다시 실행됩니다.

이 최초 이전을 한 번 수행한 뒤에는 앱 안의 `업데이트 확인` 버튼을 사용할 수 있습니다. 기존 설정과 `TOTAL_CLASSIFIED`는 그대로 유지됩니다.

## 소스 버전 사용자

`master` 브랜치의 소스 버전은 기존과 같이 `update.bat`으로 업데이트합니다. EXE 자동 업데이트 코드는 `exe-release` 브랜치에서만 관리합니다.

## Lite 빌드 참고

배포 용량을 줄이기 위해 torch/CUDA/transformers 기반 AI 자동분류 패키지는 포함하지 않습니다. 갤러리, 캔버스, 기본 분류, Real-CUGAN 업스케일 기능은 포함됩니다.
